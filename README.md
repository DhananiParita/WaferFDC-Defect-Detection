# WaferFDC-Defect-Detection
Automated Semiconductor Fault Detection &amp; Classification (FDC) system using PyTorch (ResNet-18) and Flask. Features a responsive real-time dashboard for identifying yield-killing defects like Scratches, Donuts, and Edge-Rings.
